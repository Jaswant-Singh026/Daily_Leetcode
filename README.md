# Daily_Leetcode

Welcome to my **Daily-Leetcode** repository!

This repository contains my daily solutions to LeetCode problems as part of my Data Structures & Algorithms (DSA) practice and interview preparation journey.

##  Goals

* Solve problems consistently.
* Improve problem-solving and analytical thinking.
* Strengthen DSA concepts.
* Prepare for coding interviews.

##  Repository Structure

Each solution is organized by problem and includes:

* Problem name
* Solution code
* Time Complexity
* Space Complexity (where applicable)

##  Languages

* C++ (Primary)

##  Topics Covered

* Arrays
* Strings
* Hashing
* Two Pointers
* Sliding Window
* Binary Search
* Linked Lists
* Stacks & Queues
* Trees
* Binary Search Trees
* Heaps
* Graphs
* Dynamic Programming
* Greedy Algorithms
* Backtracking
* Recursion
* Bit Manipulation
* And more...

##  Progress

I update this repository regularly as I solve new LeetCode problems.

##  Contributions

This repository is intended for learning and interview preparation. Suggestions and improvements are always welcome.
